GET /api

Current page, displays the list of possible endpoints

/api/topics

Returns an array of topics

GET /api/topics/:slug

Returns an array of topics, specified by a slug

GET /api/topics/:slug/articles

Returns an array of articles, specified by a slug

GET /api/articles

Returns an array of articles

GET /api/articles/:article_id

Returns a single article, specified by an article id

PATCH /api/articles/:article_id?votes="up"

Returns a single article, specified by an article id, with an incremented vote count

PATCH /api/articles/:article_id?votes="down"

Returns a single article, specified by an article id, with a decremented vote count

GET /api/articles/:article_id/comments

Returns an array of comments for an article, specified by an article id

POST /api/articles/:article_id/comments

Adds and returns a new comment, to an article, specified by an article id

GET /api/comments

Returns an array of comments

PATCH /api/comments/:comment_id?votes="up"

Returns a single comment, specified by a comment id, with an incremented vote count

PATCH /api/comments/:comment_id?votes="down"

Returns a single comment, specified by a comment id, with an decremented vote count

DELETE /api/comments/:comment_id

Deletes a single comment, specified by a comment id

GET /api/users

Returns an array of users

GET /api/users/:username

Returns an array of users, specified by a username
