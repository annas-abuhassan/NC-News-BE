/api

GET /topics => Returns an array of topics

GET /topics/:slug => Returns an array of topics, specified by a slug

GET /topics/:slug/articles => Returns an array of articles, specified by a slug

GET /articles => Returns an array of articles

GET /articles/:article_id => Returns a single article, specified by an article id

PATCH /articles/:article_id?votes="up" => Returns a single article, specified by an article id, with an incremented vote count

PATCH /articles/:article_id?votes="down" => Returns a single article, specified by an article id, with a decremented vote count

GET /articles/:article_id/comments => Returns an array of comments for an article, specified by an article id

POST /articles/:article_id/comments => Adds and returns a new comment, to an article, specified by an article id

GET /comments => Returns an array of comments

PATCH /comments/:comment_id?votes="up" => Returns a single comment, specified by a comment id, with an incremented vote count

PATCH /comments/:comment_id?votes="down" => Returns a single comment, specified by a comment id, with an decremented vote count

DELETE /comments/:comment_id => Deletes a single comment, specified by a comment id

GET /users => Returns an array of users

GET /users/:username => Returns an array of users, specified by a username
